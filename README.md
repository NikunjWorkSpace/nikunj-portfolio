# Nikunj Patel - Portfolio Website

A modern, responsive portfolio website built with HTML5, CSS3, and JavaScript.

## 🌟 Features

- **Modern Design**: Dark theme with gradient accents
- **Responsive**: Mobile-first approach, works on all devices
- **Smooth Animations**: Engaging animations and transitions
- **Interactive Elements**: Smooth scrolling, hover effects
- **Contact Form**: Functional contact section
- **SEO Optimized**: Meta tags and semantic HTML

## 📁 File Structure

```
nikunj-portfolio/
├── index.html      # Main HTML file
├── styles.css      # Styling and animations
├── script.js       # JavaScript functionality
└── README.md       # Documentation
```

## 🚀 How to Deploy on Netlify

### Step 1: Push to GitHub
```bash
git add .
git commit -m "Initial portfolio deployment"
git push origin main
```

### Step 2: Connect to Netlify
1. Go to [netlify.com](https://netlify.com)
2. Click **"Sign up with GitHub"**
3. Authorize Netlify to access your GitHub
4. Click **"New site from Git"**
5. Select your **`nikunj-portfolio`** repository
6. Click **"Deploy site"**

## ⚙️ Customization

### Update Your Information

Edit `index.html` and replace:
- `Nikunj Patel` with your name
- `nikunj@example.com` with your email
- `+1 (234) 567-890` with your phone
- Project descriptions with your actual projects
- Add your GitHub, LinkedIn, Twitter links

### Change Colors

Edit `styles.css` and modify the `:root` variables:
```css
:root {
    --primary-color: #6366f1;      /* Main purple */
    --secondary-color: #ec4899;    /* Pink accent */
    --dark-bg: #0f172a;            /* Dark background */
}
```
